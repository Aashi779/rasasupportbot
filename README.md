# rasasupportbot

### Create a virtual environment
`pip install rasa`

### Activate virtual environment

### Train model
`rasa train`

### Run rasa in terminal
`rasa shell`

### Run RASA on web (tested only on M1 mac chrome)
`rasa run --enable-api --cors="*"`
Open `index.html` in your browser.
